# LeadFinder — Dashboard

A full-stack SaaS dashboard for identifying local businesses with a weak digital presence and generating AI-powered outreach messages.

## ✦ Technologies

- React
- Vite
- Recharts
- Axios

## 🚀 Features

- Search businesses by city and category
- Visualize business distribution with interactive charts
- Live lead summary statistics
  - Total leads
  - Weak leads
  - Businesses without websites
  - Average rating
- Business cards with weakness indicators
  - No website
  - No Instagram
  - Low review count
- AI-generated outreach messages for weak leads
- Color-coded UI for quick lead assessment
- Integrated with a live REST API backend

## 🛠 The Process

I built this dashboard to provide a clean and intuitive way to analyze lead data generated by my LeadFinder backend API.

The project started with designing a simple and consistent UI using a violet and rose color palette. I focused on creating reusable React components and built each section independently, including the Navbar, SearchBar, Business Cards, Charts, and Statistics panels before combining everything into a single dashboard experience.

After completing the frontend structure, I connected the application to the deployed Railway backend using Axios for API communication. The application was then deployed on Vercel with automatic redeployment enabled through GitHub integration.

## 💡 What I Learned

- React component architecture and reusable components
- State management using React hooks
- Asynchronous data fetching with Axios
- Data visualization using Recharts
- Integrating a React frontend with a deployed REST API
- Deploying Vite applications on Vercel
- Working with a full-stack monorepo project structure

## 🖥 Running the Project

Clone the repository:

```bash
git clone https://github.com/vishnupriya-shisode/leadfinder-frontend.git
```

Install dependencies:

```bash
npm install
```

Update the API base URL in:

```javascript
src/services/api.js
```

Example:

```javascript
const API_BASE = "http://localhost:8000";
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

in your browser.
