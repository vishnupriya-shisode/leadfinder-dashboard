# Local Business Opportunity Dashboard

## Project Idea

A dashboard-based SaaS tool that helps identify local businesses with weak digital presence such as missing websites, low social activity, or poor ratings. The platform organizes business data visually and can later use AI to generate outreach suggestions automatically.

---

## Goal

Build a beginner-friendly full-stack SaaS project that teaches:
- backend systems
- APIs
- databases
- AI integration
- dashboards
- deployment
- real-world product workflows

---

## MVP Features

- Search businesses by city/category
- Store business data
- Display businesses in dashboard
- Detect:
  - no website
  - low ratings
  - missing Instagram
- Basic analytics
- AI-generated outreach suggestions
